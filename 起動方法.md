# 起動方法

このプロジェクトのWebサイトをパソコン上で動かすための手順です。
ターミナル（Macなら「ターミナル」アプリ）の操作に慣れていない人向けに、細かく手順を書いています。

## 1. 必要なソフトをインストールする

以下の2つが必要です。すでに入っている場合はこの手順は飛ばして大丈夫です。

### Node.js（Webサイトを動かすための土台となるソフト）

1. [Node.js公式サイト](https://nodejs.org/) を開く
2. 「LTS」と書かれているバージョンをダウンロードしてインストールする（**v22.22.0以上が必須**）

インストールできたか確認するには、ターミナルで以下を実行します。

```
node --version
```

`v22.22.3` のようにバージョン番号が表示されればOKです（v22.22.0未満の場合はインストールし直してください）。

### pnpm（このプロジェクトで使うパッケージ管理ツール）

ターミナルで以下を実行してインストールします。

```
npm install -g pnpm
```

確認方法：

```
pnpm --version
```

`9.15.0` のようにバージョン番号が表示されればOKです。

## 2. プロジェクトを手元にダウンロードする（初回のみ）

すでにフォルダを持っている場合はこの手順は不要です。

```
git clone https://github.com/itizawa/roadside-station.git
cd roadside-station
```

## 3. プロジェクトのフォルダに移動する

ターミナルを開いて、このプロジェクトのフォルダに移動します。
（Finderでフォルダを開いて、ターミナルにドラッグ＆ドロップすると `cd` の後にパスを自動で入力できます）

```
cd roadside-station
```

## 4. 必要な部品（ライブラリ）をインストールする（初回のみ）

```
pnpm install:website
```

これでWebサイトの動作に必要なファイルが `website/node_modules` にダウンロードされます。
少し時間がかかることがあります。

## 5. 開発用サーバーを起動する

```
pnpm dev
```

実行すると、以下のような表示が出ます。

```
  ➜  Local:   http://localhost:5173/
```

表示された `http://localhost:5173/` （または似たようなURL）をブラウザで開くと、サイトが確認できます。

止めるときは、ターミナルで `Ctrl + C` を押します。

## 6. 本番用にビルドする（サイトを公開する前の最終確認）

```
pnpm build
```

`website/build/client` フォルダに、公開用のファイル一式が作られます。

ビルドした内容をブラウザで確認したいときは、以下を実行します。

```
pnpm preview
```

## よくあるトラブル

- **`command not found: pnpm` と表示される**
  → 手順1の pnpm インストールができていません。`npm install -g pnpm` を再度実行してください。
- **`pnpm dev` を実行してもすぐ終了してしまう / エラーが出る**
  → 手順4の `pnpm install:website` を実行し忘れていないか確認してください。
- **ポート（5173など）がすでに使われていると言われる**
  → 他のターミナルで `pnpm dev` を起動したままになっていないか確認してください。`Ctrl + C` で止めてから再実行してください。

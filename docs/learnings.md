# 学習メモ：リサーチ観点・台本フィードバックの蓄積

`research-station`（トリビア候補の選定）と `script-station`（台本のブラッシュアップ）が、過去の人間の判断から学んだ傾向を記録する場所。

- 両スキルは実行の最初にこのファイルを読み込み、記載された傾向を踏まえて候補出し・台本生成を行う
- 実行の最後に、今回の判断（採用/不採用・フィードバック内容）を「個別ログ」に追記する
- 同じ種類の判断が複数回（目安2回以上）確認できたときだけ、上の要約セクション（「採用されやすい観点」等）を更新する。1回限りの判断は個別ログに残すだけに留め、無理に一般化しない

## リサーチ：トリビア候補の選定傾向

### 採用されやすい観点

- 数字を伴う劇的な「ビフォーアフター」「転換」のストーリー（例：ワースト1位→返上、衰退→復活、傍流の副産物→国内トップシェア）。単なる由来・出身地紹介より好まれる（しょうなん・桶川で2回確認）

### 不採用になりやすい観点

- 地名由来・人物の出身地・建立年・行政の節目（周年記念）など、変化を伴わない静的な事実紹介（しょうなん・桶川で2回確認）

#### 2026-07-26_しょうなん
- 採用：手賀沼が27年連続で全国水質汚濁ワースト1だったが、道の駅開業とほぼ同時期（2001年）に返上したという歴史（理由の言及なし）
- 不採用：沼南町の名称由来／白樺派文人の手賀沼移住／「てんと」棟の命名・受賞歴／マツコの知らない世界で紹介の野菜バーガー（理由の言及なし）

#### 2026-08-01_桶川
- 採用：紅花栽培が平成に一度「復活」した歴史（桶川ロータリークラブが平成5〜8年にかけて山形から種を譲り受け試験栽培→べに花まつり開催、理由：ユーザーから「平成以降・桶川市固有の話題」の指定あり）
- 採用：釣りエサ国内トップシェアメーカー「マルキユー株式会社」（桶川市赤堀に本社、養蚕業の副産物からの産業転換）（理由：ユーザーから「意外性のある話題が良い」との明確な指定あり）
- 不採用：皇女和宮の宿泊／桶川宿の宿場町規模／徳川秀忠・新選組が通った道／泉福寺／熊野神社古墳／紅花商人寄進の石燈篭／牧野康成の陣屋（理由：ユーザーから「情報が古い、平成以降がいい」との指摘で除外）
- 不採用：俳優・本木雅弘の出身地／旧陸軍桶川飛行学校跡地（平和祈念館）／市制55周年／マンホールカード（オケちゃん）／圏央道桶川北本IC開通／赤堀工業団地の地名由来／昼夜間人口比率のギャップ（理由：「意外性のある話題」の基準でマルキユーより弱いと判断され選ばれず。昼夜間人口比率は一次資料への到達不足という裏取りの弱さも一因）
- 不採用：桶川ストーカー殺人事件（理由：全国的に有名な平成の出来事だが、動画のトーン（落ち着いた散歩・グルメ紹介）に合わないとAssistant判断で候補提示時点から除外。ユーザーからの明示的な却下ではない）

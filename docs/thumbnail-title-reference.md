# サムネイル・タイトル参考事例（2026-07-05）

拡散されるかどうかはサムネイルと切り口次第、という前提のもと、参考になりそうな動画を分析した記録。今後サムネイル・タイトルを作るときの型として参照する。

## 分析対象

- タイトル：「誰も行かないガチ秘境''群馬県の南西部''を全力で観光してみた！！」
- チャンネル：ひろき / 世界旅行（登録者29.2万人）
- 実績：454,661回視聴（2025/01/25公開、約1年で45万回）
- 動画尺：1時間54分・チャプター21個
- URL：https://youtu.be/OE74ZchXVbE

## サムネイルの型

- **地図を主役にした構図**：対象エリア全体の衛星写真（Googleマップ風）を中央に配置し、目的地だけを赤い縁取りで囲んで「←ここ」という矢印付きテキストで指し示す
- **左右の帯に大きな文字**：左＝「誰も行かない」（白背景に黒/赤の太字）、右＝「群馬の奥地」（黒背景に白文字、アクセント部分だけ黄色）
- **配色は白・黒・赤・黄の4色のみ**に絞り、地図の情報量と文字の視認性を両立させている

→ 道の駅シリーズでも「関東のどこにあるか」を地図で一目で伝え、対象の道の駅だけを赤枠＋矢印で強調する構図は流用しやすい。

## タイトルの型

- 「誰も行かない」「ガチ秘境」という強い断定・煽り言葉で希少性を打ち出す
- 「''群馬県の南西部''」のように**地域を具体的に絞り込む**ことで、県名だけより「本当に人が行かなそう」という説得力が増す
- 「全力で観光してみた！！」で行動の熱量・エンタメ性を付与

→ 「知られざる」「歩いてしか行けない」のような希少性ワード＋具体的な地域名の組み合わせが応用できそう。

## 説明欄・チャプターの型

- 冒頭にビジネス連絡先・SNSリンクを固定配置
- 21個の詳細チャプター（地名・スポット名を明記）で、1時間54分の長尺でも視聴者が目的の場面へ飛べるようにしている
- キーワードタグに地名・ジャンル語を列挙（例：ひろき,旅行,世界旅行,鉄道,ひとり旅,日航,群馬県）

→ `script.md`の構成を考える際、チャプター単位で見どころを明示する前提で台本を組むと使い回しやすい。

## ショート動画との連動

検索結果には本編に加えて「群馬の秘境地 至福のひととき」「群馬が誇る秘境の温泉5選」など、**同エリアの見どころを1テーマずつ切り出したショート動画**が並んでいた。本編とショートで同じ「秘境」ワールドを多面的に展開し、複数の入り口から本編へ誘導していると考えられる。

→ 道の駅の回でも、本編とは別に「見どころ1つだけ」を切り出したショートを作ることで、拡散経路を増やせる可能性がある。

# サムネイル・タイトル参考事例（2026-07-05）

拡散されるかどうかはサムネイルと切り口次第、という前提のもと、参考になりそうな動画を分析した記録。今後サムネイル・タイトルを作るときの型として参照する。

## 分析対象

- タイトル：「誰も行かないガチ秘境''群馬県の南西部''を全力で観光してみた！！」
- チャンネル：ひろき / 世界旅行（登録者29.2万人）
- 実績：454,661回視聴（2025/01/25公開、約1年で45万回）
- 動画尺：1時間54分・チャプター21個
- URL：https://youtu.be/OE74ZchXVbE

## サムネイルの型

- **地図を主役にした構図**：対象エリア全体の衛星写真（Googleマップ風）を中央に配置し、目的地だけを赤い縁取りで囲んで「←ここ」という矢印付きテキストで指し示す
- **左右の帯に大きな文字**：左＝「誰も行かない」（白背景に黒/赤の太字）、右＝「群馬の奥地」（黒背景に白文字、アクセント部分だけ黄色）
- **配色は白・黒・赤・黄の4色のみ**に絞り、地図の情報量と文字の視認性を両立させている

→ 道の駅シリーズでも「関東のどこにあるか」を地図で一目で伝え、対象の道の駅だけを赤枠＋矢印で強調する構図は流用しやすい。

### 地図画像の権利・取得方法について

Google Mapsの地図・航空写真は著作権があり、YouTube動画（教育・エンタメ目的）では画面内の帰属表示（"Map data ©2026 Google"等）を残せば無許可利用できるが、サムネイルのようにトリミングして帰属表示が消えるケースは規約上グレー。

代替として国土地理院の「地理院タイル」（`https://cyberjapandata.gsi.go.jp/xyz/...`）は認証不要・出典表示のみで商用利用も可だが、1タイル256×256pxの正方形画像しか返らないため、群馬の動画のような広域1枚絵を作るには複数タイルの合成が必要で手間がかかる。OSM系の「キー不要で範囲指定→1枚画像」を返す静的地図サービス（staticmap.openstreetmap.de等）も調査したが、現在はサービス終了・アクセス制限で実用的な選択肢がなかった。

**結論：ブラウザで地図サイト（OpenStreetMap.orgや地理院地図）を開き、該当エリアにズームしてスクリーンショットを撮る方法を採用する。** 週1回の頻度であればAPIキー取得や合成スクリプトの整備は過剰で、Claude Codeのブラウザ操作機能でその都度スクショを撮るのが最も手軽。

## タイトルの型

- 「誰も行かない」「ガチ秘境」という強い断定・煽り言葉で希少性を打ち出す
- 「''群馬県の南西部''」のように**地域を具体的に絞り込む**ことで、県名だけより「本当に人が行かなそう」という説得力が増す
- 「全力で観光してみた！！」で行動の熱量・エンタメ性を付与

→ 「知られざる」「歩いてしか行けない」のような希少性ワード＋具体的な地域名の組み合わせが応用できそう。

## 説明欄・チャプターの型

- 冒頭にビジネス連絡先・SNSリンクを固定配置
- 21個の詳細チャプター（地名・スポット名を明記）で、1時間54分の長尺でも視聴者が目的の場面へ飛べるようにしている
- キーワードタグに地名・ジャンル語を列挙（例：ひろき,旅行,世界旅行,鉄道,ひとり旅,日航,群馬県）

→ `script.md`の構成を考える際、チャプター単位で見どころを明示する前提で台本を組むと使い回しやすい。

## ショート動画との連動

検索結果には本編に加えて「群馬の秘境地 至福のひととき」「群馬が誇る秘境の温泉5選」など、**同エリアの見どころを1テーマずつ切り出したショート動画**が並んでいた。本編とショートで同じ「秘境」ワールドを多面的に展開し、複数の入り口から本編へ誘導していると考えられる。

→ 道の駅の回でも、本編とは別に「見どころ1つだけ」を切り出したショートを作ることで、拡散経路を増やせる可能性がある。

## タイトルの型（横断的な傾向）

1. **数字の可視化が圧倒的多数**：「〇選」「全〇駅制覇」「TOP20」「33位→2位」のように件数・順位を必ず数字で明記する。「なんとなく良かった」ではなく「定量的にすごい」ことを一目で伝える構造。
2. **権威付けワード**：「マニアが選んだ」「道の駅連絡会公認」「元日本代表が調査」など、第三者的な信頼性・専門性を付与する語がタイトル/企画に頻出。
3. **地域名＋最新性のセット**：都道府県名・エリア名を明記しつつ「【最新版】」「2025年版」を添え、検索性（地域名で引っかかる）と鮮度訴求を両立させている。
4. **達成・制覇の物語化**：単なる紹介ではなく「3泊4日で」「半年かけて」など期間を明示した"挑戦ストーリー"としてタイトル化すると再生が伸びる傾向。「全19駅制覇」のように完遂の達成感を前面に出す。
5. **グルメの主役化**：「マジで食べてほしい」「絶品」のような口語強調＋具体的な品目数（「4選」）の組み合わせが多い。

## サムネイルの型（横断的な傾向）

- 施設の外観写真よりも**食べ物のアップ**を主役にするパターンが目立つ（グルメ系・ランキング系とも）。
- ランキング/選抜系は**大きな数字**（「TOP20」「5選」「33位→2位」）を画面内に太字で配置し、文字だけで内容が伝わる設計。
- 車中泊・車旅系は**車＋道の駅の看板／夜景**を組み合わせた実写サムネが定番（車を持たない本チャンネルの読者には距離があるため、直接の流用は不向き）。
- ローカル局・情報番組系は**タレント/専門家の顔出し＋「プロと初心者」のような対比構造**をサムネで見せ、権威付けと親しみやすさを両立。

## 本チャンネル（徒歩×道の駅）への示唆

- 「道の駅特化チャンネル」の先行事例（だんぺいとスーザン）は登録者1.3万人規模に留まっており、道の駅単体テーマは市場としてはニッチ。前回のconcept.md分析どおり、「散歩・街歩き」文脈を主軸にする方針の妥当性が裏付けられた。
- 他チャンネルの主流である「車中泊」「〇駅制覇」は車前提の企画のため、本チャンネルの"徒歩縛り"とは訴求軸が異なる。むしろ**「車の道の駅動画には出てこない、駅から歩いて行くまでの生活道路・商店街・史跡」という道中パートを主役に据える差別化**が有効。
- 一方で「数字の可視化」「達成の物語化」は縛りプレイ性（徒歩90分以内ルール）と相性が良く、流用できる：
  - 例：「最寄り駅から徒歩◯分」を地図サムネ内に大きく明記する／タイトルに具体的な分数を入れる（「電車で来て、あとは徒歩78分だけ。道の駅◯◯」等）
  - 「グルメの主役化」はそのまま流用可能（特産品・グルメ紹介パートのサムネ候補カットとして、館内で撮る食べ物アップを必ず1カット確保する）
  - チャプター分けは既存の道の駅系動画でも標準的な手法であり、`script.md`の5パート構成（オープニング／徒歩パート／館内紹介／グルメ紹介／エンディング）をそのままYouTubeチャプターに変換する運用でよい。

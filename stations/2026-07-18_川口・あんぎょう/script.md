# 台本：道の駅 川口・あんぎょう

## タイトル・サムネイル案

- **採用タイトル**：【徒歩1時間】都心のすぐそばに、まさかの"植木の里"があった
  - 「車がなくても道の駅に行ける」という企画の核＋「都会→植木の里」のギャップを両立
  - ※実移動時間は徒歩32〜40分＋電車13分＋乗換待ちで正味45〜55分（route.md参照）。「徒歩1時間」は体感時間としての表現なので、サムネ・台詞では「歩いて」「電車を挟んで」等の言い回しで実態との乖離を補足する
- **採用サムネイル：ルート地図型（人物カットなし・マップ＋文字のみ）**
  - 赤羽駅→戸塚安行駅→道の駅までの矢印付き地図を画面全体のベースに配置
  - 顔カット・人物写真は使わない
  - デカ文字コピー：「徒歩で行ける道の駅」よりさらに短く・大きく（6〜8文字目安、太めフォント・2〜3色に抑える）。小さいサムネ表示でも一瞬で読めることを優先する
  - 地図と文字だけで"徒歩ルートの存在"を一目で伝え、他の道の駅チャンネル（車前提）との差別化ポイントを視覚化する

## 動画の目的・伝えたいこと

- **切り口：「なんで都会に近い川口に道の駅が！？」** 道の駅＝地方の国道沿い・山間部というイメージを裏切り、都心から電車で1時間かからない川口市に道の駅がある意外性をフックにする
- 種明かしは、道の駅として作られたのではなく、もともとあった地域産業施設「川口緑化センター」（植木・園芸・造園の拠点）に道の駅の指定が後から加わったという珍しい成り立ちと、都心至近・住宅密集地という立地の意外性
  - ※道の駅は制度上もともと「休憩機能」「情報発信機能」「地域連携（地域振興）機能」の3つを併せ持つのが全国共通のルール（1993年の登録制度開始時から）。「休憩所ではなく地域振興拠点」という対比は不正確なので使わない
- 徒歩パートで少しずつ住宅街から植木屋・園芸畑が増えていく風景の変化を使い、都会から植木の里へ切り替わっていく様子を視覚的にも見せる

## 構成

### 1. オープニング（駅到着〜出発）

- 赤羽駅からスタート。「車がなくても、歩いて行ける道の駅があるって知ってました？」と問いかけ、サムネイルのルート地図と対応させる形で"徒歩で行ける"ことをまず提示する
- この時点では道の駅の正体（植木の里）は明かさず、伏線として残す

### 2. 徒歩パート（道中の紹介）

- 赤羽駅→赤羽岩淵駅の徒歩区間：赤羽一番街商店街など「都会らしい」街並みを見せる
- 埼玉高速鉄道で戸塚安行駅へ移動する電車内で、川口市＝鋳物の街というイメージを軽く振っておく（「キューポラのある街」は1962年の古い映画に由来する呼び名で、今では知らない人も多い豆知識として紹介する程度に留め、一般常識であるかのようには扱わない）
- 戸塚安行駅から道の駅までの徒歩区間で本題を展開：
  - 「実は川口には鋳物と並ぶもう一つの伝統産業がある」と切り出し、植木・園芸・造園業＝「植木の里 安行」の話へ
  - 承応年間（1652〜55年）に吉田権之丞が栽培を始め、明暦の大火の復興需要で江戸に売り込んだのが始まり、という起源エピソードを紹介
  - 関東大震災後、駒込・巣鴨の植木職人が移住して盆栽の産地としても発展した話
  - 徒歩中に見える植木屋・園芸畑の実際の景色と重ねてナレーションする（研究段階では西福寺・金剛寺など周辺スポットの位置関係も参照）
  - 「だからこの道の駅は、都心至近の住宅街のど真ん中にありながら、もともとあった地域産業施設『川口緑化センター』に道の駅の指定が後から加わってできた」という着地で、オープニングの伏線を回収

### 3. 道の駅到着〜館内紹介

- 川口緑化センター「樹里安」の館内（園芸売店、JA直売所、和菓子売店）を紹介
- 2025年オープンの新レストラン「Cafe Restaurant ANDG」で施設の新しい一面も見せる

### 4. グルメ・特産品紹介

- 名物「樹里安アイス」（安行寒桜・山椒・柚子の3種）
- ANDGのボタニカル料理

### 5. エンディング（まとめ・次回予告）

- 「都会のすぐそばに、400年続く植木の里があった」という気づきで締める
- 鋳物の街・川口のもう一つの顔として印象づける

## 台詞メモ

- 冒頭フック：「車がなくても、歩いて行ける道の駅があるって知ってました？」
- キューポラの豆知識（軽く触れる程度）：「川口は昔から鋳物の街で、1962年には『キューポラのある街』という映画の舞台にもなりました。今ではあまり知られていない話かもしれませんが、実はもう一つ、鋳物と並ぶ伝統産業がこの街にはあるんです」
- 歴史パート導入：「その名も“植木の里 安行”。始まりは今から約370年前、江戸時代の承応年間まで遡ります」
- 着地：「都心からすぐの住宅街のど真ん中。しかもこの道の駅、最初から道の駅として作られたわけじゃなくて、もともとあった植木・造園の拠点施設に、道の駅の指定が後から加わってできたものなんです」
